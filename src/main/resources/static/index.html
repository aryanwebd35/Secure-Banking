About SecureBank

A full-stack digital banking platform built with modern Java & Spring ecosystem, demonstrating enterprise-grade security, real-time transactions, and clean architecture.

Tech Stack

Backend: Spring Boot 4.1, Spring Security, Spring Data JPA, Hibernate ORM
Database: MySQL 9.0 with Hibernate auto-DDL
Auth: Clerk (email OTP + Google) → JWT tokens (jjwt 0.11) + Spring Security
Frontend: Vanilla HTML5, CSS3, ES6+ JavaScript (no framework)
Security: BCrypt password hashing, CORS config, stateless sessions
APIs: RESTful endpoints with OpenAPI / Swagger UI

Features
  • Clerk-powered authentication with email OTP — separate sign-in and sign-up flows
  • User-provided phone number onboarding — used as unique transfer ID
  • Deposit & Withdraw with full transaction history
  • Peer-to-peer instant transfer via phone number
  • Admin dashboard with account management (block/unblock/close)
  • JWT-based stateless authentication with role-based access
  • Global exception handling with proper HTTP status codes
  • Real-time balance and transaction tracking
